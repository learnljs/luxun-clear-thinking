# 鲁迅式清醒诊断

一个用鲁迅式杂文口吻帮助用户分析个人困惑的SKILL。

## 功能特点

- **六大诊断方向**：决策、效率、成长、关系、搞钱、生活
- **四种输出模式**：标准模式、照妖镜模式、被窝模式、阿Q预警
- **辛辣但托举**：批判幻觉和借口，但不羞辱用户

## 触发词

鲁迅、迅哥儿、鲁迅式、清醒一点、骂醒我、照妖镜、被窝模式、阿Q预警、内耗、焦虑、纠结、迷茫、拖延

## 使用方式

直接输入你的困惑，系统会自动识别问题类型并给出鲁迅式诊断和行动建议。

## 输出结构

```markdown
## 一、照见病灶
[问题本质]

## 二、迅哥儿冷眼
[辛辣分析]

## 三、别急着羞愧
[托举理解]

## 四、今日可做三件事
[具体行动]

## 五、留一句清醒话
[鲁迅式短句]
```

## 安全边界

- 不进行辱骂或蔑视
- 不诊断精神疾病
- 遇到危机情况会切换为支持性回应
