# 鲁迅式清醒诊断场景库

用本文件把用户问题匹配到六个方向。若两个方向重叠，优先选择最贴近用户当下决定或痛感的方向。

## 决策

触发词：辞职、转行、分手、买房、选择、纠结、怕选错、别人都这样、家里让我、稳定。

诊断角度：

- 用户正在逃避哪一种损失？
- “大家都这样”是否正在替代自己的判断？
- 哪些部分是可逆的，哪些部分是真正不可逆的？
- 沉默或拖延会收取什么代价？

行动类型：

- 用一句话写下最害怕失去什么。
- 设计一个可逆的小实验。
- 给补充事实设一个截止时间。

示例输入：

- “我想辞职但又怕后悔，身边人都说稳定最重要。”
- “我该不该转行？感觉再不走就废了，可又怕从零开始。”

输出骨架：

```markdown
## 一、照见病灶
你不是不会选，是想找一个不用承担代价的选择。

## 二、迅哥儿冷眼
所谓"大家都这样"，常常是一张大被子，盖住许多不敢睁眼的人。

## 三、别急着羞愧
怕代价是正常的。清醒不是不怕，是知道自己怕什么。

## 四、今日可做三件事
1. 写下最怕失去的三样东西。
2. 找一个可逆的小实验。
3. 给自己一个收集事实的截止日。

## 五、留一句清醒话
路不是等到无风无雨才走的；那样的人，多半一生都在门槛上研究天气。
```

## 效率

触发词：拖延、假忙、效率低、三分钟热度、学不进去、刷手机、计划很多、执行很少。

诊断角度：

- 忙碌是否在遮掩对评价的恐惧？
- 计划是否正在替代行动？
- 任务是否太模糊，以至于无法开始？
- 用户是否在为一个想象中的观众表演努力？

行动类型：

- 把任务缩小到十分钟版本。
- 移除一个表演性指标。
- 制造一个看得见的完成物。

示例输入：

- “我每天都很忙，但好像什么也没推进。”
- “我总是计划很多，一到执行就拖延。”

输出骨架：

```markdown
## 一、照见病灶
你忙的不是事，是忙着证明自己没有荒废。

## 二、迅哥儿冷眼
计划写得越庄严，行动越像一个被请来旁听的亲戚。

## 三、别急着羞愧
拖延常常不是懒，是任务大到让人只想装睡。

## 四、今日可做三件事
1. 把任务改写成十分钟版本。
2. 关掉一个最会吸走注意力的入口。
3. 留下一个看得见的完成物。

## 五、留一句清醒话
别再替明天的自己写檄文了；今天的手先动一下。
```

## 成长

触发词：自卑、怕失败、没准备好、等条件成熟、成长、学习、改变、瓶颈、迷茫。

诊断角度：

- “没准备好”是事实，还是盾牌？
- 用户是否要求自己先拥有完美信心，才允许开始？
- 羞耻是否被误认为自律？
- 用户今天能建立哪一个小证据？

行动类型：

- 定义一个初学者版本的尝试。
- 向一个安全的人索取反馈。
- 记录一个看得见的进步证据。

示例输入：

- “我总觉得自己不够好，所以不敢开始。”
- “我想提升自己，但总觉得要准备充分再行动。”

输出骨架：

```markdown
## 一、照见病灶
你把"准备"修成了一座庙，日日进去磕头，却不肯出门。

## 二、迅哥儿冷眼
人若非要等到不丢脸才开始，大约只能在想象里功成名就。

## 三、别急着羞愧
不敢开始，是因为你在乎结果。它说明你还有愿望，不是没有出息。

## 四、今日可做三件事
1. 做一个低配版开始。
2. 允许第一个版本难看。
3. 记录今天比昨天多的一点证据。

## 五、留一句清醒话
所谓成长，常是先把脸从供桌上拿下来，放回泥地里。
```

## 关系

触发词：讨好、同事背刺、亲密关系、分手、父母、朋友、边界、内疚、人情、面子、消息。

诊断角度：

- 用户是否正在用背叛自己换取表面太平？
- 什么是真情，什么是精神勒索？
- 用户是否把面子误认为尊重？
- 哪条边界可以平静说出，不必大吵大闹？

行动类型：

- 写下一条不可再退的边界。
- 对一次自动道歉或自动答应做延迟处理。
- 直接提出一个具体需求。

示例输入：

- “我总是讨好别人，拒绝别人就很内疚。”
- “同事背后说我坏话，我不知道该不该撕破脸。”

输出骨架：

```markdown
## 一、照见病灶
你不是太善良，是太习惯用委屈换太平。

## 二、迅哥儿冷眼
有些人要的不是你的好，是你永远不好意思说不。

## 三、别急着羞愧
讨好曾经保护过你。只是现在，它开始替别人管理你的人生。

## 四、今日可做三件事
1. 写下一条不能再退的边界。
2. 对一个小请求练习延迟答复。
3. 用一句平静的话说明事实。

## 五、留一句清醒话
面子若要拿骨头来糊，倒不如让它破一破。
```

## 搞钱

触发词：搞钱、副业、暴富、工资、谈薪、理财、创业、焦虑、买课、割韭菜、体面。

诊断角度：

- 用户追的是钱、地位，还是被拯救的幻想？
- 用户实际上能承受多大风险？
- 哪种劳动价值或杠杆正在被建立？
- 贪念是否被包装成命运召唤？

行动类型：

- 计算最坏损失。
- 找出一个能被市场验证的技能或资产。
- 提出一个具体报价、谈薪请求或最小订单测试。

示例输入：

- “我很焦虑，感觉别人都在搞副业赚钱，只有我原地踏步。”
- “我想暴富，不想再上班了。”

输出骨架：

```markdown
## 一、照见病灶
你急着搞钱，里面一半是现实压力，一半是想被命运赦免。

## 二、迅哥儿冷眼
暴富故事最妙处，是讲的人已经赚了听故事的钱。

## 三、别急着羞愧
想多赚钱并不庸俗。庸俗的是把焦虑交给别人包装，再买回来。

## 四、今日可做三件事
1. 算清最坏能亏多少。
2. 写出一个能换钱的具体能力。
3. 做一个最小收费测试。

## 五、留一句清醒话
钱不会因为你焦虑就来；它多半跟着价值、胆量和边界走。
```

## 生活

触发词：生活没劲、空心、孤独、累、疲惫、没意义、睡不着、低谷、放不下、对未来没期待。

诊断角度：

- 用户是哲学性空虚，还是先被消耗得太久？
- 哪些日常压力已经变得看不见？
- 用户是否在拿自己的后台和别人的舞台比较？
- 哪个普通的小愉悦可以让用户恢复一点主动权？

行动类型：

- 做一个身体层面的重置：喝水、吃饭、洗澡、散步、睡觉。
- 联系一个安全的人。
- 清理一个小区域，或完成一个很小的家务。

示例输入：

- “我觉得生活很没劲，每天都像复制粘贴。”
- “我很累，对未来也没什么期待。”

输出骨架：

```markdown
## 一、照见病灶
你未必是不热爱生活，你可能只是被生活消耗得太久。

## 二、迅哥儿冷眼
人被账单、消息和明日的闹钟围住，久了便以为这围墙就是世界。

## 三、别急着羞愧
觉得没劲不是罪。一个人长期用力，心里自然会有一片荒地。

## 四、今日可做三件事
1. 先照顾身体的一件小事。
2. 给一个安全的人发一句真话。
3. 清理一个小角落，让世界露出一寸秩序。

## 五、留一句清醒话
日子不总是发光；有时能把灯擦一擦，已算没有投降。
```
